# 📅 Turnito

🇪🇸 Descripción

Turnito es una aplicación web de gestión de turnos diseñada como una solución SaaS multi-tenant, pensada para distintos rubros como barberías, centros de estética y profesionales de la salud.

Permite a los clientes reservar turnos de forma simple, mientras que los negocios pueden administrar su agenda de manera eficiente, todo desde una única plataforma.

🚀 Funcionalidades
👤 Clientes
Ver profesionales disponibles
Consultar horarios en tiempo real
Reservar turnos fácilmente
Recibir confirmación de reservas
💼 Profesionales
Gestión de agenda personal
Configuración de disponibilidad
Visualización de turnos asignados
🏢 Empresas (Multi-tenant) (Proximamente...)
Espacio independiente para cada negocio
Aislamiento de datos por empresa
Gestión de profesionales y clientes
🛠️ Tecnologías
Frontend & Backend: Next.js (App Router)
Estilos: Tailwind CSS
Base de datos: Supabase (PostgreSQL)
Autenticación: Supabase Auth
Deploy: Vercel

🇺🇸 Description

Turnito is a web-based appointment management application designed as a multi-tenant SaaS solution for industries such as barbershops, beauty salons, and healthcare professionals.

It allows clients to easily book appointments, while businesses can efficiently manage their schedules — all in one platform.

🚀 Features
👤 Clients
View available professionals
Check real-time availability
Book appointments easily
Receive booking confirmations
💼 Professionals
Personal schedule management
Availability configuration
View assigned appointments
🏢 Businesses (Multi-tenant) (Comming soon...)
Dedicated workspace for each business
Data isolation per company
Manage professionals and clients
🛠️ Tech Stack
Frontend & Backend: Next.js (App Router)
Styling: Tailwind CSS
Database: Supabase (PostgreSQL)
Authentication: Supabase Auth
Deployment: Vercel
