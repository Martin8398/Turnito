# 📅 Turnito

ES

Turnito es una aplicación web de gestión de turnos pensada como solución SaaS para múltiples rubros (barberías, centros de estética, profesionales de la salud, etc.).

Permite a los clientes reservar turnos de forma simple y a los negocios administrar su agenda de manera eficiente, todo en una misma plataforma.

# 🚀 Features
# 👤 Clientes
Ver profesionales disponibles
Consultar horarios en tiempo real
Reservar turnos fácilmente
Confirmación de reservas
💼 Profesionales
Gestión de agenda personal
Configuración de disponibilidad
Visualización de turnos asignados
🏢 Empresas (multi-tenant)
Cada negocio tiene su propio espacio
Aislamiento de datos por empresa
Gestión de profesionales y clientes
🛠️ Tech Stack
Frontend & Backend: Next.js (App Router)
Estilos: Tailwind CSS
Base de datos: Supabase (PostgreSQL)
Autenticación: Supabase Auth
Deploy: Vercel

EN

Turnito is a web-based appointment management application designed as a SaaS solution for various sectors (barbershops, beauty salons, healthcare professionals, etc.).

It allows clients to easily book appointments and businesses to efficiently manage their schedules, all on a single platform.

🚀 Features 👤 Clients: View available professionals; Check real-time schedules; Easily book appointments; Booking confirmations 💼 Professionals: Personal schedule management; Availability settings; View assigned appointments 🏢 Businesses (multi-tenant): Each business has its own space; Data isolation per company; Professional and client management 🛠️ Tech Stack: Frontend & Backend: Next.js (App Router); Styles: Tailwind CSS; Database: Supabase (PostgreSQL); Authentication: Supabase Auth; Deployment: Vercel
